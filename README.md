# Storage Contract

## Storage

`Storage` — абстракция для работы с Key-Value данными.  
Контракт не зависит от реализации и сообщает **только о внутренних ошибках хранилища**.


### Set(key string, value []byte) error

Записывает значение по ключу.

- если ключ существует — значение перезаписывается
- возвращает `error` только при внутренней ошибке

### Get(key string) (value []byte, err error)

Получает значение по ключу.

- если ключ существует — возвращает `(value, nil)`
- если ключ не существует — возвращает `(nil, nil)`
- возвращает `error` только при внутренней ошибке

### Delete(key string) error

Удаляет значение по ключу.

- если ключ не существует — ничего не делает
- возвращает `error` только при внутренней ошибке

## Гарантии

- thread-safe
- отсутствие ключа не считается ошибкой
- методы не паникуют
- внутренняя реализация скрыта
